# Changelog

## [1.0.0] - 2026-02-27

### Added
- Initial release of Vent Mode plugin
- Vent detection skill with 5 tone categories (Sarcastic, Self-aware, Empathetic, Dramatic, Competitive)
- UserPromptSubmit hook with lightweight bash classifier
- SessionStart hook for automatic activation
- `/vent-mode` slash command (status, demo, examples)
- Escalation system for consecutive vents
- Safety rails: control commands, technical messages, and long messages always pass through
